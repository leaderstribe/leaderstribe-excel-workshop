# Session 5: Excel Tables

**1. Session Title:** Excel Tables

**2. Learning Goal:** Learners can convert a plain data range into a proper Excel Table and understand why Tables are more reliable than plain ranges.

**3. Duration:** 2 hours

**4. Facilitator Preparation:** Use the cleaned version of `l1_savings_group_contributions.csv` from Session 4.

**5. Materials Needed:** Laptop per learner, cleaned dataset from Session 4, projector.

**6. Sample Dataset:** Cleaned `l1_savings_group_contributions.csv`.

**7. Live Demonstration Steps**
1. Select the data range and press Ctrl+T (or Insert > Table).
2. Confirm "My table has headers."
3. Show the new banded rows, filter arrows, and Table Design tab.
4. Rename the Table (e.g. "SavingsData") in the Table Name box.
5. Add a new row at the bottom and show it auto-formats.
6. Add a Total Row and show it can Sum the Contribution column automatically.

**8. Guided Hands-On Activity**
Together: convert the dataset to a Table, rename it, and add a Total Row summing contributions.

**9. Independent Practice Task**
Learners add 3 new rows of their own invented data to the bottom of the Table and confirm the Total Row updates automatically.

**10. Expected Output**
A named Excel Table with a working Total Row that updates when new data is added.

**11. Common Mistakes and Fixes**
- *Mistake:* Typing new data just below the Table instead of inside it. *Fix:* Show the small blue corner handle that marks the Table's boundary — new rows must go inside it.
- *Mistake:* Believing Tables and Tables' formatting can't be changed. *Fix:* Show Table Design > Table Styles to pick a different look.
- *Mistake:* Confusing an Excel Table with a "table" in the everyday sense (any grid of data). *Fix:* Clarify: an Excel Table is a *named, structured* object with special powers (auto-expand, structured references, Total Row).

**12. Reflection Questions**
- How does a Table help when your savings group adds new members every month?
- What changed automatically when you added new rows?

**13. Assignment**
Convert one of your own real lists (attendance, contributions, sales) into a Table before the next session.

**14. Facilitator Notes**
Tables are the foundation for Level 2 (PivotTables, lookups). Do not rush this session — a strong grasp here makes Level 2 much easier.
