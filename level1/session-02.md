# Session 2: Data Entry & Formatting

**1. Session Title:** Data Entry & Formatting

**2. Learning Goal:** Learners can enter data accurately, format text and numbers, adjust column widths/row heights, and make a sheet easy to read.

**3. Duration:** 2 hours

**4. Facilitator Preparation**
- Load `l1_market_sales.csv` onto every laptop.
- Prepare an example of a "messy" vs "well-formatted" sheet to show side by side.

**5. Materials Needed:** Laptop per learner, `l1_market_sales.csv`, projector.

**6. Sample Dataset:** `l1_market_sales.csv` — daily market sales records from traders in Onitsha, Kano, and Kejetia markets.

**7. Live Demonstration Steps**
1. Open `l1_market_sales.csv`. Show how numbers align right and text aligns left by default — explain why (Excel is telling you what it thinks each cell is).
2. Bold the header row and apply a background fill colour.
3. Widen columns using AutoFit (double-click the column border).
4. Format the "Total Sales (NGN)" column as Currency with no decimal places.
5. Format the "Date" column consistently using Format Cells.
6. Freeze the header row (View > Freeze Panes) so it stays visible while scrolling.

**8. Guided Hands-On Activity**
Together: bold and colour the header row, freeze it, and format the Unit Price column as currency.

**9. Independent Practice Task**
Learners format the Total Sales column as currency, centre-align the Quantity Sold column, and apply a light fill colour to every other row for readability.

**10. Expected Output**
A neatly formatted market sales sheet: bold headers, frozen top row, currency formatting, and alternating row colours.

**11. Common Mistakes and Fixes**
- *Mistake:* Typing "5,000" as text instead of a number. *Fix:* Show how text-numbers align left; retype using Format Cells > Number.
- *Mistake:* Formatting numbers as currency changes the value. *Fix:* Reassure learners formatting only changes appearance, not the underlying number.
- *Mistake:* Freeze Panes freezing the wrong row. *Fix:* Click the cell just below the row you want frozen before applying Freeze Panes.

**12. Reflection Questions**
- Why does Excel treat numbers and text differently?
- How does good formatting help someone reading your sheet for the first time?

**13. Assignment**
Format a small list of 10 items from home (e.g. household shopping list) with bold headers and currency formatting.

**14. Facilitator Notes**
Emphasise that formatting is not decoration — it's communication. A well-formatted sheet is taken more seriously by supervisors and donors.
