# Session 1: Excel Interface & Workbook Navigation

**1. Session Title:** Excel Interface & Workbook Navigation

**2. Learning Goal:** By the end of this session, each learner can open Excel, identify the main parts of the screen, move around a worksheet, and save a workbook correctly.

**3. Duration:** 2 hours

**4. Facilitator Preparation**
- Test that Excel opens on every laptop before learners arrive.
- Load `l1_attendance_register.csv` onto every laptop or a shared drive.
- Prepare a printed "map" of the Excel screen (Ribbon, Tabs, Cell, Row, Column, Name Box, Formula Bar) to hand out — many learners find labelled diagrams reassuring.

**5. Materials Needed**
- Laptop per learner (or pair)
- `l1_attendance_register.csv`
- Printed screen-map handout
- Projector for live demo

**6. Sample Dataset:** `l1_attendance_register.csv` — a simple workshop attendance register with learner names, states, and attendance marks across 4 sessions.

**7. Live Demonstration Steps**
1. Open Excel and open a **new blank workbook**. Point out the Ribbon, Tabs (Home, Insert, etc.), the grid of rows and columns, the Name Box, and the Formula Bar.
2. Show how a cell is named by its column letter and row number (e.g. cell B3).
3. Click on different cells and show the Name Box changing.
4. Use arrow keys, Tab, and Enter to move between cells.
5. Open `l1_attendance_register.csv` and show how a workbook can have multiple **sheets** (tabs at the bottom).
6. Rename a sheet tab and change its colour.
7. Save the file using **Save As**, choosing a clear file name and location, in `.xlsx` format.

**8. Guided Hands-On Activity**
Together, as a group: open `l1_attendance_register.csv`, click on the cell containing "Halima Yusuf," then use the Name Box to jump to cell D5. Everyone renames Sheet1 to "Attendance."

**9. Independent Practice Task**
Each learner opens a new blank workbook, renames Sheet1 to her own first name, types her name in cell A1, and saves the file as `[HerName]_Session1.xlsx` in a folder she can find again.

**10. Expected Output**
A saved `.xlsx` file with one renamed sheet containing the learner's name in cell A1.

**11. Common Mistakes and Fixes**
- *Mistake:* Learner saves as `.csv` instead of `.xlsx` and loses formatting later. *Fix:* When using Save As, point out the "Save as type" dropdown and confirm it says Excel Workbook (*.xlsx).
- *Mistake:* Learner double-clicks and accidentally moves a cell's content. *Fix:* Show Ctrl+Z (Undo) immediately — tell learners Ctrl+Z is their best friend.
- *Mistake:* Learner cannot find where she saved the file. *Fix:* Always save to a labelled Desktop folder created at the start of the workshop, e.g. "Excel Workshop 2026."

**12. Reflection Questions**
- What part of the Excel screen was the most confusing at first?
- Can you now find a specific cell if I tell you its name (e.g. "go to C4")?

**13. Assignment**
Before next session: open Excel at home or on a friend's laptop and practice moving between at least 10 different cells using both the mouse and the arrow keys.

**14. Facilitator Notes**
This session sets the emotional tone for the whole workshop. Do not rush. Some learners will need one-on-one help just to open Excel. Pair a faster learner with a slower one — this builds community and confidence on both sides.
