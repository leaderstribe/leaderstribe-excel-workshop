# Session 3: Sorting & Filtering

**1. Session Title:** Sorting & Filtering

**2. Learning Goal:** Learners can sort data by one or more columns and use filters to view only the rows they need.

**3. Duration:** 2 hours

**4. Facilitator Preparation:** Load `l1_savings_group_contributions.csv`. Prepare 3 sample questions learners must answer using sort/filter (e.g. "Who contributed the least in Week 2?").

**5. Materials Needed:** Laptop per learner, `l1_savings_group_contributions.csv`, projector.

**6. Sample Dataset:** `l1_savings_group_contributions.csv` — weekly savings (ajo/esusu-style) contributions from a women's savings group across several towns.

**7. Live Demonstration Steps**
1. Select the data range and apply Sort (Data tab > Sort) by "Contribution (NGN)" from smallest to largest.
2. Show Sort by multiple levels: Town (A–Z), then Contribution (largest to smallest).
3. Apply Filter (Data tab > Filter) and show the dropdown arrows appear on headers.
4. Filter to show only Week 2 rows.
5. Filter to show only "Cash" payment method.
6. Clear all filters.

**8. Guided Hands-On Activity**
Together: sort the sheet by Town then Week. Then filter to show only members from Lagos.

**9. Independent Practice Task**
Learners must find and write down: (a) who paid the most in Week 3, (b) how many members used Mobile Money, (c) all rows where Notes is not blank.

**10. Expected Output**
Correct answers to the 3 questions, achieved using Sort and Filter (not by manually scanning).

**11. Common Mistakes and Fixes**
- *Mistake:* Selecting only one column before sorting, which scrambles the data. *Fix:* Always select the whole table, or use "Expand the selection" when prompted.
- *Mistake:* Forgetting a filter is still active, causing confusion later. *Fix:* Show the small funnel icon that appears on filtered columns as a visual reminder.
- *Mistake:* Sorting a Table with formulas incorrectly. *Fix:* Explain sort works safely within Excel Tables (covered in Session 5).

**12. Reflection Questions**
- How could sorting help you prepare a report for your savings group treasurer?
- What is the difference between sorting and filtering?

**13. Assignment**
Bring a small list of numbers/names from home or work and practice sorting it two different ways.

**14. Facilitator Notes**
Use a real savings group scenario ("Who owes contribution this week?") to make the skill click emotionally, not just technically.
