# Session 8: Conditional Formatting, Basic Reports & Capstone Launch

**1. Session Title:** Conditional Formatting, Basic Reports & Capstone: "Everyday Data Story"

**2. Learning Goal:** Learners can apply conditional formatting to highlight important data, assemble a simple one-page report, and complete their Level 1 capstone project.

**3. Duration:** 3 hours (longer session — includes capstone work time and mini-presentations)

**4. Facilitator Preparation:** Review the [Level 1 Capstone Guide](../capstone/level1-capstone.md) and [Assessment Rubric](../rubrics.md) before this session. Ask learners in Session 7 to bring a small real dataset from their own life or work.

**5. Materials Needed:** Laptop per learner, learners' own small datasets (or `l1_women_enterprise_directory.csv` as backup), projector, printed rubric.

**6. Sample Dataset:** `l1_women_enterprise_directory.csv` (backup/demo) or learner's own dataset.

**7. Live Demonstration Steps**
1. Apply Conditional Formatting > Highlight Cell Rules > Greater Than, to highlight contributions above a chosen value in green.
2. Apply a Colour Scale to a numeric column to show a "heat map" style view.
3. Apply an Icon Set (e.g. arrows) to show up/down trends.
4. Assemble a simple one-page report: Table + one chart + one summary sentence typed in a cell, e.g. "Total sales this week: [formula result]."
5. Use Page Layout > Print Area and Page Setup to make the report print-ready on one page.

**8. Guided Hands-On Activity**
Together: apply a colour scale to the Years Active column in `l1_women_enterprise_directory.csv`, and build one combined report view (table + chart + one summary sentence).

**9. Independent Practice Task — Capstone Work Time**
Learners spend the rest of the session on their **"Everyday Data Story"** capstone: clean their own dataset (using Session 4 skills), format it (Session 2), sort/filter it (Session 3), build a Table (Session 5), add at least one formula (Session 6), one IF classification (Session 7), one chart (Session 7), and one conditional formatting rule (this session).

**10. Expected Output**
A completed one-page "Everyday Data Story": a clean, Table-based, formatted sheet with at least one chart, one IF formula, and one conditional formatting rule, plus a one-sentence written summary of what the data shows.

**11. Common Mistakes and Fixes**
- *Mistake:* Applying too many conditional formatting rules, making the sheet confusing. *Fix:* Recommend no more than 1–2 rules per report for clarity.
- *Mistake:* Running out of time because cleaning wasn't done early. *Fix:* Remind learners to clean data first — everything else depends on it.
- *Mistake:* Capstone dataset is too large/complicated for the time available. *Fix:* Encourage learners to use a subset (e.g. one month, one location) if their real data is too big.

**12. Reflection Questions**
- What is the one story your data tells that you didn't notice before you cleaned and charted it?
- Which Level 1 skill do you think you'll use the most in real life?

**13. Assignment**
Finalise and save the capstone file as `[HerName]_EverydayDataStory.xlsx`, ready to present or submit. Bring it to the graduation/showcase session.

**14. Facilitator Notes**
This is a celebration session, not a test. Let 2–3 volunteers briefly show their "Everyday Data Story" to the group. Use the [Assessment Rubric](../rubrics.md) to give warm, specific, constructive feedback — not just a score.
