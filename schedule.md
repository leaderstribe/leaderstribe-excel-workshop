# Full 8-Week Schedule

The program runs as **two 8-session levels**, typically one session per week (8 weeks per level), or condensed into an intensive 2-week bootcamp (one session per day). Each session is 2–2.5 hours.

## Level 1 — Beginner (Weeks 1–8)

| Week | Session | Topic |
|---|---|---|
| 1 | [Session 1](level1/session-01.md) | Excel Interface & Workbook Navigation |
| 2 | [Session 2](level1/session-02.md) | Data Entry & Formatting |
| 3 | [Session 3](level1/session-03.md) | Sorting & Filtering |
| 4 | [Session 4](level1/session-04.md) | Basic Data Cleaning |
| 5 | [Session 5](level1/session-05.md) | Excel Tables |
| 6 | [Session 6](level1/session-06.md) | Basic Formulas & Functions |
| 7 | [Session 7](level1/session-07.md) | IF Statements & Simple Charts |
| 8 | [Session 8](level1/session-08.md) | Conditional Formatting, Basic Reports & Capstone: "Everyday Data Story" |

## Level 2 — Intermediate (Weeks 9–16)

| Week | Session | Topic |
|---|---|---|
| 9 | [Session 1](level2/session-01.md) | Remove Duplicates & Text-to-Columns |
| 10 | [Session 2](level2/session-02.md) | Flash Fill & Data Validation / Dropdowns |
| 11 | [Session 3](level2/session-03.md) | VLOOKUP & XLOOKUP |
| 12 | [Session 4](level2/session-04.md) | INDEX-MATCH |
| 13 | [Session 5](level2/session-05.md) | COUNTIF, SUMIF, AVERAGEIF, IFS, IFERROR |
| 14 | [Session 6](level2/session-06.md) | PivotTables |
| 15 | [Session 7](level2/session-07.md) | PivotCharts & Slicers |
| 16 | [Session 8](level2/session-08.md) | Dashboards & Capstone: "Community Data in Action" |

## Notes for facilitators

- Learners must complete Level 1 (or demonstrate equivalent skill) before starting Level 2.
- Build in a **make-up session** or two — attendance in community workshops is rarely 100%.
- Weeks 8 and 16 run longer (add 30–45 minutes) because of capstone presentations.
