# Session 7: PivotCharts & Slicers

**1. Session Title:** PivotCharts & Slicers

**2. Learning Goal:** Learners can turn a PivotTable into a visual PivotChart and add Slicers for easy, button-based filtering.

**3. Duration:** 2.5 hours

**4. Facilitator Preparation:** Use the PivotTable built in Session 6 from `l2_cooperative_sales_data.csv` (or rebuild fresh at the start of class).

**5. Materials Needed:** Laptop per learner, `l2_cooperative_sales_data.csv`, projector.

**6. Sample Dataset:** `l2_cooperative_sales_data.csv`.

**7. Live Demonstration Steps**
1. Click inside the existing PivotTable, go to PivotTable Analyze > PivotChart, choose a Column Chart.
2. Show how the chart updates instantly if the PivotTable's fields change.
3. Go to PivotTable Analyze > Insert Slicer, select "Region" and "Product."
4. Click different Slicer buttons and show both the PivotTable and PivotChart updating together.
5. Format the Slicer (resize, choose a colour) so it's easy to use for a non-technical audience.

**8. Guided Hands-On Activity**
Together: build a PivotChart showing Units Sold by Month, add a Region Slicer, and test filtering to "Mombasa" only.

**9. Independent Practice Task**
Learners add a Product Slicer to their own Session 6 PivotTable/Chart and test filtering to a single product.

**10. Expected Output**
A PivotChart connected to at least one working Slicer that filters both the chart and the table together.

**11. Common Mistakes and Fixes**
- *Mistake:* Slicer appears not to filter the chart. *Fix:* Confirm the Slicer is connected to the same PivotTable the chart is based on (Slicer > Report Connections).
- *Mistake:* Chart type chosen doesn't fit the data (e.g. Pie chart with too many categories, becoming unreadable). *Fix:* Recommend Column charts for comparisons across more than 4–5 categories.
- *Mistake:* Learner resizes the Slicer awkwardly, covering the chart. *Fix:* Show drag-handles to resize and reposition calmly.

**12. Reflection Questions**
- How would a Slicer help a cooperative leader explore sales data herself, without needing you to rebuild the report each time?
- What's the difference between a normal chart (Session 7, Level 1) and a PivotChart?

**13. Assignment**
Practice building one PivotChart + Slicer combination using your own PivotTable data.

**14. Facilitator Notes**
This session often produces genuine delight — learners see how a "self-service" report feels. Emphasise this is exactly what Session 8's Dashboard will bring together.
