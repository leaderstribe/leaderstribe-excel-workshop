# Level 2 — Intermediate Track Overview

## Who this is for
Women who have completed Level 1 (or already know the basics: entering data, formatting, sorting/filtering, basic formulas). Level 2 moves into the tools that let a learner turn messy real-world data into decisions — the kind of Excel skill that gets you noticed at an NGO, cooperative, or small business.

## Goal of Level 2
By the end of Level 2, a learner should be able to clean a messy dataset (duplicates, split columns, Flash Fill), build dropdown-controlled data entry, look up information across sheets (VLOOKUP, XLOOKUP, INDEX-MATCH), summarise data with conditional functions (COUNTIF, SUMIF, AVERAGEIF, IFS, IFERROR), build a PivotTable and PivotChart, filter with Slicers, and assemble it all into a simple one-page Dashboard.

## Capstone: "Community Data in Action"
Each learner (or small team) takes a realistic, messier community dataset (NGO beneficiary list, cooperative sales records, microloan repayments) and delivers a cleaned dataset, at least two lookup/summary formulas, one PivotTable + PivotChart, and a one-page dashboard with a Slicer. See the [Level 2 Capstone Guide](../capstone/level2-capstone.md).

## Facilitator note
Level 2 learners can get discouraged when formulas don't work on the first try. Normalise the `#N/A` and `#VALUE!` errors early — tell them on Day 1 that **every Excel user sees these every day**, including you.
