# Assessment Rubrics

Use these rubrics for capstone grading and for informal session-by-session progress checks. Scores are out of 4 per criterion: **1 = Not yet, 2 = Developing, 3 = Meets expectation, 4 = Strong / helping others.**

## Level 1 Capstone: "Everyday Data Story"

| Criterion | 1 | 2 | 3 | 4 |
|---|---|---|---|---|
| Data cleaning | Data still has obvious errors | Some cleaning done, inconsistent | Data mostly clean and consistent | Fully clean; learner explains her cleaning decisions |
| Formatting | Hard to read, no formatting | Basic formatting only | Clear headers, currency/number formats correct | Polished, reader-friendly, print-ready |
| Table use | Plain range, no Table | Table created but underused | Table with working Total Row | Table used confidently, e.g. new rows added and auto-updating |
| Formulas | No working formulas | Formula present but incorrect | At least one correct SUM/AVERAGE/MIN/MAX/COUNT | Multiple correct formulas, confidently explained |
| IF logic | Missing or broken | Attempted, minor errors | Correct, simple IF formula | Correct IF formula plus clear real-world reasoning |
| Chart | Missing or unreadable | Chart present, poorly labelled | Clear, correctly labelled chart | Chart clearly tells a story, well-chosen chart type |
| Conditional formatting | Missing | Applied but unclear purpose | One clear, useful rule | Thoughtful use that highlights something meaningful |
| Verbal/written summary | Missing | Vague | One clear insight stated | Insight stated with a suggested next action |

## Level 2 Capstone: "Community Data in Action"

| Criterion | 1 | 2 | 3 | 4 |
|---|---|---|---|---|
| Duplicate handling | Not attempted | Attempted, unclear logic | Duplicates correctly identified and removed | Clear documented decision on what counts as duplicate |
| Data validation | Missing | Dropdown present but broken | Working dropdown with correct list | Dropdown includes helpful input/error messages |
| Lookup formula | Missing/broken | Attempted, errors present | Correct VLOOKUP/XLOOKUP/INDEX-MATCH | Correct, with IFERROR handling and clear explanation |
| Conditional summary formulas | Missing | Partially correct | Correct COUNTIF/SUMIF/AVERAGEIF/IFS | Multiple correct formulas, well chosen for the story |
| PivotTable | Missing | Built but confusing/incorrect | Clear, correct 2-dimension summary | Well-structured, renamed fields, clean layout |
| PivotChart & Slicer | Missing | One present, not connected | Both present and working together | Polished, intuitive, clearly connected to decision-making |
| Dashboard assembly | Missing | Cluttered or incomplete | Clean one-page layout, functional | Professional-looking, clear KPI(s), tells a full story |
| Presentation | Missing | Unclear | Clear walkthrough of the dashboard | Confidently explains the decision the data supports |

## Using these rubrics well

- Share the rubric with learners **before** capstone work begins (Session 7 or the start of Session 8), not as a surprise at the end.
- Always pair a score with **one specific strength and one specific next step** — avoid a bare number.
- A learner scoring mostly 2s has still completed the workshop successfully — treat the rubric as a growth map, not a pass/fail gate.
