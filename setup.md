# Workshop Setup Guide

Read this at least **one week before** the workshop begins.

## 1. Software

- **Microsoft Excel** (2016 or newer, or Microsoft 365) on every learner's laptop.
  - If a learner only has a phone: the free **Excel mobile app** works for Level 1 sessions 1–4, but Level 2 (PivotTables, Slicers) needs a laptop or desktop.
  - No Excel license? Google Sheets can substitute for Level 1, but tell learners clearly that some button names will look different.
- Confirm every laptop opens Excel **before** Day 1. Do not assume — test it in a WhatsApp group chat the week before.

## 2. Room / venue

- Power sockets for every 2 learners (extension boards are your best friend — expect NEPA/power cuts, so a generator or inverter backup is strongly recommended).
- Tables where learners can see each other's screens for pair work.
- A projector or large screen for live demonstrations.
- Internet access is a "nice to have," not a requirement — all datasets are provided offline as CSV files.

## 3. Materials checklist

- [ ] One laptop per learner (or per pair, minimum)
- [ ] Practice dataset CSVs loaded onto every laptop (from the [datasets folder](datasets/))
- [ ] Printed or shared copy of the [Practice Workbook Structure](workbook-structure.md)
- [ ] Attendance sheet
- [ ] Name tags (first names only, large and bold — many learners are shy to ask for names again)
- [ ] Extension boards / power strips
- [ ] Projector + adapter cables (HDMI and VGA — don't assume which one the venue has)
- [ ] Printed certificates template for graduation day

## 4. Before Day 1: facilitator prep

1. Test every session's live demo yourself, step by step, on the actual dataset — don't just read the script.
2. Print or share the [Assessment Rubrics](rubrics.md) so you know what "success" looks like before you teach.
3. Set up a WhatsApp or SMS group for the cohort for reminders and encouragement between sessions.
4. Decide your teaching pace: Level 1 assumes **zero prior Excel experience**. It is normal to spend extra time on Session 1.

## 5. Facilitator mindset

Many learners in this program have been told, directly or indirectly, that "data" and "computers" are not for them. Your job in the first hour is not to teach Excel — it's to prove that wrong. Be patient. Celebrate small wins loudly. No question is a stupid question.
