# Practice Datasets

All datasets are plain CSV files, chosen for realistic African community, small-business, and NGO contexts. Download and open directly in Excel.

## Level 1 (Beginner) datasets

| File | Used in | Description |
|---|---|---|
| [l1_attendance_register.csv](l1_attendance_register.csv) | Session 1 | Workshop attendance register across 4 sessions |
| [l1_savings_group_contributions.csv](l1_savings_group_contributions.csv) | Sessions 3, 4, 6, 7 | Weekly ajo/esusu-style savings group contributions, with intentional inconsistencies for cleaning practice |
| [l1_market_sales.csv](l1_market_sales.csv) | Sessions 2, 6 | Daily market trader sales from Onitsha, Kano, and Kejetia markets |
| [l1_women_enterprise_directory.csv](l1_women_enterprise_directory.csv) | Session 8 | Directory of small women-led businesses across several sectors |

## Level 2 (Intermediate) datasets

| File | Used in | Description |
|---|---|---|
| [l2_ngo_beneficiaries_duplicates.csv](l2_ngo_beneficiaries_duplicates.csv) | Session 1 | NGO beneficiary list with genuine duplicate entries |
| [l2_full_names_to_split.csv](l2_full_names_to_split.csv) | Sessions 1, 2 | Names stored as "Surname Firstname" for Text-to-Columns / Flash Fill practice |
| [l2_community_health_data.csv](l2_community_health_data.csv) | General practice | Community health visit records across three areas |
| [l2_cooperative_sales_data.csv](l2_cooperative_sales_data.csv) | Sessions 3, 4, 6, 7, 8 | Sales orders from women's cooperatives in Nairobi and Mombasa |
| [l2_lookup_reference_table.csv](l2_lookup_reference_table.csv) | Sessions 3, 4 | Clean reference table of product codes, categories, and prices |
| [l2_microloan_repayments.csv](l2_microloan_repayments.csv) | Sessions 2, 5, 8 | Microloan repayment tracker for women-owned small businesses |
